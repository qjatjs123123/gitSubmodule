# samil-dev-kit (sparse-checkout test repo)

## 목적

sparse-checkout **no-cone 모드**로 `TextEditor` 폴더를 선택적으로 제외하는 테스트를 위한 레포입니다.

## 폴더 구조

| 폴더 | 설명 |
|------|------|
| `TextEditor/` | 텍스트 에디터 관련 컴포넌트 (sparse-checkout 제외 테스트 대상) |
| `components/` | 공통 UI 컴포넌트 모음 |
| `hooks/` | 커스텀 React 훅 모음 |
| `stories/` | Storybook 스토리 파일 모음 |
| `src/` | 메인 소스 코드 |
| `utils/` | 유틸리티 함수 모음 |
| `types/` | TypeScript 타입 정의 |
| `styles/` | 전역 스타일 및 테마 |
| `constants/` | 상수 값 정의 |
| `assets/` | 이미지, 폰트 등 정적 자산 |
